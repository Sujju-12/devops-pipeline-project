# DevOps CI/CD Pipeline Project

## Architecture
End-to-end DevOps pipeline for a Java Spring Boot microservice.

## Process
I am doing this project with support from Claude Ai and Chatgpt with no coding. I have copied the code and built project files from claude. My main goal from this project is to support java spring boot microservice with DevOps workflow making it fully automated when once the developer pushes the code. 

## Tools Used
- **Git** — Version Control
- **Maven** — Build & Test
- **Jenkins** — CI/CD Orchestration
- **Docker** — Containerization
- **Terraform** — Infrastructure as Code
- **Ansible** — Configuration Management
- **Kubernetes (Minikube)** — Container Orchestration
- **Prometheus + Grafana** — Monitoring & Alerting
- **SonarQube** — Code Quality
- **Trivy** — Security Scanning

## Pipeline Flow
`Code Push → GitHub → Jenkins → Maven Build → Docker Image → K8s Deploy → Monitor`
